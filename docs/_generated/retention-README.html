Retention · Overview

Retention — make it stick (the "woken at 2am" test)

You've read the kit. Reading is not learning. This folder is where you prove the system lives in your head — so you could explain it cold, with no notes, even if someone woke you at 2am.

If the rest of the kit was the lecture, this is the lab and the exam. Everything here is built on one research-backed idea: you remember what you struggle to recall, not what you re-read.

Why active recall (and not re-reading)

Re-reading feels productive — the words look familiar, so you assume you know them. That feeling is a trap called fluency: recognizing a sentence is not the same as being able to produce it.

Active recall flips it. Instead of looking at the answer, you try to retrieve it from memory first — and only then check. The effort of retrieval is what physically strengthens the memory. A wrong attempt that you then correct beats a smooth re-read every time.

That is why every answer on every page here is hidden behind a fold:

Example — try to answer before you open this

Opening this fold after you've guessed is the whole technique. If you opened it without guessing, you got a re-read, not a recall. Close it, guess, then check.

Analogy

re-reading is watching someone else lift weights. Recall is lifting them yourself. Only one builds muscle.

Spaced repetition — beat the forgetting curve

You forget most of what you learn within days unless you revisit it. Spaced repetition means reviewing material at growing intervals — right before you'd forget it. Each successful recall pushes the next review further out.

  1. Read the kit in order (start at the primer).
  2. Self-test — every page already ends with a ✅ Check yourself block. Answer before opening.
  3. Flashcards — one Q→A fold per term/concept. Retrieve, then flip.
  4. Exercises — hands-on drills: trace a run, decode a regex, gate an if:, spot a bug. Doing, not reading.
  5. Capstone — the "explain it back" assessment, completed unaided.
  6. Graduation checklist — the concrete bar to be signed off.

How to use each file

  • Flashcards: cover the answer. Say or write your answer out loud / on paper. Then open the fold. If you were wrong or vague, that card comes back sooner.
  • Exercises: actually do the task (open the real file, run the real command) before peeking at the hint. Open the solution last, to grade yourself.
  • Capstone: no notes, no folds open. Write your answers first, then check.
The one rule

Guess before you peek. A fold you open without first attempting the answer taught you nothing.
